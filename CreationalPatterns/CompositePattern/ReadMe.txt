What..?
	Composite Design Pattern states that “Compose objects into tree structures to represent 
	part-whole hierarchies.
	Composite let clients treat individual objects and compositions of objects uniformly. 

Why..?
	Composite pattern is used where we need to treat a group of objects in similar way as a single object.
	Composite pattern composes objects in term of a tree structure to represent part as well as whole 
	hierarchy.
